Schedule Documentation

Admin will be able to create a schedule view from uploaded CSVs or inputted data. Admin will be able to check and edit schedules for constraint integrity


    • Admin will be able to create schedules using created sections, preferences, and sets.
    • To navigate to the schedule maker:
      • Click [Begin Scheduling Process]
      • Choose schedule from dropdown. These schedule names were created in Step 3 for manual input and Step 6 in using imported data
      • Choose the constraint set in dropdown
      • Click [Run]
    • Admin will be able to view and edit schedule to account for constraint integrity. Admin will be able toggle classes on and off
    • If a class time is scheduled in which there exists a violation of a preference or constraint, the cell will be marked in red. Admin has the ability to edit the timing of the course and change the class settings as needed
    • To save view as CSV, click the button at the bottom of the screen
